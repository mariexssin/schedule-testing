# Чеклісти для тестування

## Модуль: Перегляд завантаженості аудиторій

### Функціональність
- [ ] Перевірка відображення таблиці для конкретного корпусу.
- [ ] Перевірка коректності кольору при завантаженості 0%.
- [ ] Перевірка переходу кольору в жовтий при 50% зайнятості.
- [ ] Перевірка переходу кольору в червоний при 91% зайнятості.
- [ ] Перевірка відображення тултіпа при наведенні курсору.
- [ ] Перевірка точності даних у тултіпі відповідно до таблиці.
- [ ] Відображення парного/непарного тижня.

### Негативні сценарії
- [ ] Спроба завантажити розклад для корпусу без аудиторій.
- [ ] Перевірка відображення при відсутності інтернету (Offline mode).
- [ ] Перевірка поведінки системи при спробі відправити форму без обраного дня тижня або типу тижня (нульове значення).
- [ ] Перевірка відображення таблиці при нульовій кількості пар.

### UI/UX
- [ ] Перевірка горизонтального скролу при великій кількості аудиторій.
- [ ] Перевірка контрастності тексту на червоному та жовтому фонах.
- [ ] Перевірка, чи не перекриває тултіп важливі кнопки навігації.
