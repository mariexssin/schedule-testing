# Чеклісти для тестування

## Модуль: Перегляд завантаженості аудиторій

### Функціональність
- [ ] Відображення зеленого кольору та тултіпа при повному збігу (Один викладач, одна назва та одна група/R1).
- [ ] Відображення зеленого кольору при збігу викладача та назви предмета (R2).
- [ ] Відображення жовтого кольору, якщо викладач один, але назва предмета відрізняється (R3).
- [ ] Відображення червоного кольору, якщо параметри викладача не збігаються (R4, R5, R6).
- [ ] Перевірка зміни індикації при переході між корпусами через фільтр.
- [ ] Перевірка відображення індикації для парного/непарного тижня.
- [ ] Оновлення кольорів у реальному часі після синхронізації з розкладом.
### Негативні сценарії
- [ ] Перевірка реакції системи на некоректні дані з бази.
- [ ] Перевірка поведінки системи, якщо для обраного тижня розклад ще не заповнений.
- [ ] Спроба перегляду таблиці завантаженості без відповідних прав доступу адміністратора.
- [ ] Перевірка поведінки інтерфейсу при втраті інтернет-з'єднання під час завантаження таблиці.
- [ ] Перевірка відсутності тултіпа, якщо курсор знаходиться поза межами активної комірки.
### UI/UX
- [ ] Поява тултіпа при наведенні на кольорову комірку.
- [ ] Читабельність тексту тултіпа на фоні різних кольорів (контрастність).
- [ ] Збереження коректного відображення кольорів при зміні розміру (width) стовпців таблиці.
- [ ] Адаптивність таблиці: коректне відображення сітки завантаженості на екранах з низькою роздільною здатністю.
